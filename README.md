# Dental Appointment Booking System

A modern appointment booking application with:

- Google Calendar integration
- Real-time availability checking
- Double-booking prevention
- Email notifications
- n8n workflow automation
- Responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- n8n
- Google Calendar API
- Gmail API

## Features

- Appointment scheduling
- Availability validation
- Automatic calendar event creation
- Confirmation emails
- Clinic notification emails
- Appointment summary
- Notes field support
